# chan9yu-blog

A personal blog built with Next.js, TypeScript, and TailwindCSS.

<br />

## Getting Started

1. Install dependencies

```bash
pnpm install
```

2. Start the development server

```bash
pnpm dev
```

3. Lint and type check

```bash
pnpm lint
pnpm type-check
```

4. Build

```bash
pnpm build
```
