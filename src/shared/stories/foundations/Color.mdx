# Color System

blog9yu.dev의 색상 시스템은 **실제 서비스에서 사용 중인 색상**을 기반으로 체계화되었습니다.

## Architecture

```
Primitive Colors (원시 색상 팔레트)
  ↓
Semantic Colors (시맨틱 토큰 - Light/Dark 테마)
```

- **Primitive Colors**: Tailwind CSS 기본 팔레트 (Gray, Slate, Zinc, Stone, Blue, Indigo)
- **Semantic Colors**: 실제 `tokens.css`에서 사용 중인 시맨틱 매핑 (text, background, border, primary, accent, scrollbar)

## CSS Variables

실제 `tokens.css`에서 정의된 CSS 변수:

```css
:root {
	/* Brand */
	--color-primary: 59 130 246; /* blue-500 */
	--color-primary-hover: 37 99 235; /* blue-600 */

	/* Light Theme */
	--color-bg-primary: 255 255 255;
	--color-bg-secondary: 249 250 251; /* gray-50 */
	--color-text-primary: 15 23 42; /* slate-900 */
	--color-text-secondary: 71 85 105; /* slate-600 */

	/* Accent */
	--color-accent: 99 102 241; /* indigo-500 */
	--color-accent-hover: 79 70 229; /* indigo-600 */
}

html.dark {
	/* Dark Theme */
	--color-bg-primary: 9 9 11; /* zinc-950 */
	--color-text-primary: 250 250 250; /* zinc-50 */
	--color-accent: 129 140 248; /* indigo-400 */
}
```

## 사용 방법

### Tailwind Utility Classes

```tsx
<div className="bg-primary text-primary border-primary">Content</div>
```

### CSS Variables

```css
.custom {
	background: rgb(var(--color-bg-primary));
	color: rgb(var(--color-text-primary));
}
```

## Accessibility

모든 색상 조합은 **WCAG 2.1 AA 기준**을 준수합니다.

- Text (primary on bg-primary): 4.5:1 이상
- Interactive Elements: 3:1 이상
